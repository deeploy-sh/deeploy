**deeploy** is a modern, self-hosted deployment platform. Terminal-first, open source, and designed for developers who want full control over their infrastructure.

## Features

- Zero-downtime deployments
- Auto SSL via Let's Encrypt
- Instant domains via sslip.io wildcard DNS
- Terminal-first UI (TUI)
- Self-hosted, you own your data
- SQLite by default, PostgreSQL optional

## Currently Supported

- GitHub repositories (public & private)
- Docker-based deployments
- Custom domains + SSL
- Single-user mode

## Coming Soon

- GitLab & Bitbucket support
- Multi-user / Teams
- Webhooks & CI integration
- Rollback
- Resource limits
