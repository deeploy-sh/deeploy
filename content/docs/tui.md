The deeploy TUI (Terminal User Interface) is built for keyboard-driven workflows.

## Command Palette

The command palette is the fastest way to do anything. Press `Alt+P` or `Ctrl+K` to open it.

From here you can:
- Create projects and pods
- Manage domains
- Add git tokens
- Set server domain
- Switch themes
- And more

Start typing to filter commands.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Alt+P` / `Ctrl+K` | Open command palette |
| `Esc` | Go back |
| `Enter` | Select / Confirm |
| `Tab` | Next field |
| `/` | Search in lists |
| `?` | Show help |
| `t` | Open theme switcher |

## Offline Mode

If the connection to your server is lost, the TUI enters offline mode. It will automatically reconnect when the server is available again.
