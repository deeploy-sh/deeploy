# deeploy

Modern Deployment. Terminal First. Open Source.

![deeploy TUI](internal/shared/assets/img/hero.png)

## Quick Start

**Server (VPS)**
```bash
curl -fsSL https://deeploy.sh/server.sh | sudo bash
```

**TUI (your machine)**
```bash
curl -fsSL https://deeploy.sh/tui.sh | bash
```
